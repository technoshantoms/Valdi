# VSCode Debugger Test

This node project is meant to run a full integration test for the debugger in vscode. The script to run the entire test this can be found under client/src/valdi/scripts/. To run this test successfully, you'll need to connect an Android device and unlock the device, and call run_debugger_test.sh

When working on this module please set your npm registry

```
npm config set registry https://registry.npmjs.org/
```
